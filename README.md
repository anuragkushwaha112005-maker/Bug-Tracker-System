# 🐞 Bug Tracker System

A lightweight and simple web-based Bug Tracking System built using HTML, CSS, and JavaScript. This project allows users to add, manage, and delete bug reports with data stored using localStorage.

---

## 🚀 Features

- Add new bugs with title, description, and severity  
- Delete bugs easily  
- Data saved using localStorage (remains after page refresh)  
- Simple and clean user interface  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- LocalStorage  

---

## 📂 Project Structure

Bug-Tracker-System/
│
└── index.html

---

## ▶️ How to Run

1. Download or clone this repository  
2. Open the index.html file in any browser  
3. Start adding and managing bugs  

---

## 🎯 Use Case

This project simulates a basic bug tracking system used in software testing. It helps in understanding how bugs are reported, managed, and tracked.
